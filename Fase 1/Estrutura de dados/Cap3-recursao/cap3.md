### Recursão

- È quando uma função chama a si mesma.
- Toda função recursiva tem o caso base e o caso recursivo.
- Uma pilha tem duas operações :push o pop.
- Função são alocadas através de pilhas
- Todas as chamadas de função vão para uma pilha de chamadas
- A pilha de chamada ( call stack ) pode ficar muito grande e coupar muito da memória.
- Toda função recursiva é composta de, pelo menos, duas partes: caso-base e caso recursivo. O caso-base é quando a função NÃO chama a si mesma, mas retorna um valor real; já o caso recursivo, como o próprio nome indica, é onde a recursividade ocorre (a função chama a si mesma).
